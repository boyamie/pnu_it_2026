## 💻 수업 전 필수 준비 사항 (설치 가이드)
[cite_start]오늘 실습을 위해 각자의 노트북에 파이썬과 코딩 에디터(VS Code)가 설치되어 있어야 합니다. [cite: 132, 135] 아래 순서대로 천천히 따라 해보세요!

### [cite_start]1️⃣ Python 3.11 이상 설치하기 [cite: 132]
우리가 AI와 드론에게 명령을 내릴 언어인 '파이썬'을 설치합니다.

1. **다운로드:** 파이썬 공식 홈페이지([python.org/downloads](https://www.python.org/downloads/))에 접속하여 최신 버전의 Python을 다운로드합니다.
2. **설치 실행 (매우 중요 🚨):** 다운로드한 설치 파일을 실행합니다.
   * 화면 맨 아래에 있는 **`Add python.exe to PATH`** 체크박스를 **반드시 체크**해야 합니다! (체크하지 않으면 나중에 오류가 발생합니다.)
3. **설치 완료:** `Install Now`를 클릭하여 설치를 완료합니다.

### [cite_start]2️⃣ VS Code 설치하기 [cite: 135]
파이썬 코드를 편하게 작성할 수 있게 해주는 에디터인 'Visual Studio Code'를 설치합니다.

1. **다운로드:** VS Code 홈페이지([code.visualstudio.com](https://code.visualstudio.com/))에 접속하여 파란색 `Download for Windows` (또는 Mac) 버튼을 눌러 다운로드합니다.
2. **설치 실행:** 다운로드한 파일을 실행하고, '다음'을 계속 눌러 설치를 완료합니다.

### [cite_start]3️⃣ VS Code 확장 프로그램(Extension) 설치 [cite: 135]
VS Code에서 파이썬을 더 똑똑하게 사용할 수 있도록 도와주는 추가 기능을 설치합니다.

1. VS Code를 실행합니다.
2. 화면 왼쪽 메뉴에서 **블록 4개가 모여있는 아이콘(Extensions)**을 클릭합니다.
3. 검색창에 아래의 이름들을 검색하고 **`Install(설치)`** 버튼을 누릅니다.
   * `Python` (Microsoft에서 만든 것)
   * `Korean Language Pack` (VS Code를 한국어로 사용하고 싶다면 설치 후 재시작)
