M-01. 필수 도구: Jupyter Notebook

우리가 앞으로 코드를 작성하고 실습할 Jupyter Notebook의 핵심 특징입니다.

M-04. 파이썬 기초 복습: 제어문 (조건/반복)

프로그램의 흐름을 조종하는 핵심입니다.

  • 조건문 (Conditionals): if / elif / else를 사용하여 조건에 따라 다른 코드를 실행합니다. (드론: 사람을 보면 추적, 아니면 순찰)
  • 반복문 (Loops): for 또는 while을 사용하여 코드를 여러 번 실행합니다. (예: 리스트의 모든 데이터 출력)
# 반복문 예시
drones =
for drone in drones:
    if drone == "Tello":
        print("Tello 비행 준비 완료")
    else:
        print(drone, "준비 중")

M-07. Numpy 기초 실습: 배열 생성 및 shape

Jupyter Notebook에 아래 코드를 직접 입력하고 결과를 확인하세요.

import numpy as np # 관례적인 이름

# 1. 리스트를 Numpy 배열로 변환
vec = np.array()
print("1차원 배열(vec):", vec)
print("배열 크기(shape):", vec.shape) # (4,) - 4개의 데이터

# 2. 2차원 배열(행렬) 생성 (이미지와 유사)
matrix = np.array([,])
print("2차원 배열(matrix):\n", matrix)
print("배열 크기(shape):", matrix.shape) # (2, 3) - 2행 3열
