M-01. 도구: Jupyter Notebook 핵심

Jupyter Notebook은 코드 작성, 실행 결과, 마크다운(설명)을 하나의 문서로 관리하는 데이터 분석의 표준 도구입니다.

🔑 핵심 모드(Mode) 및 단축키

  • Command Mode (셀 밖 상태 - 파란색 테두리): 셀 자체를 조작합니다.
    • m, m: 현재 셀을 Markdown(문서)으로 변환
    • y, y: 현재 셀을 Code(코드)로 변환
    • a (위에 셀 추가), b (아래에 셀 추가), dd (셀 삭제)
  • Edit Mode (셀 안 편집 상태 - 초록색 테두리): 코드를 타이핑합니다.
  • 공통 실행: Shift + Enter (실행 후 다음 셀로 이동)

M-03. 파이썬 기초: 변수와 메모리 구조

파이썬에서 변수 선언(예: a = [1, 2])은 단순히 값을 넣는 것이 아닙니다. 해당 리스트가 저장된 '메모리 주소값'을 변수 a에 연결(할당)하는 것입니다.

a = [1, 2, 3, 4, 5]
b = a          # b에 a의 '메모리 주소'를 복사함 (같은 곳을 바라봄)
a.append(6)    # a에 6을 추가함
print(b)       # b를 출력하면 어떻게 될까?

위 코드에서 =의 의미는 "같다"가 아니라, 우측 데이터의 메모리 주소를 좌측 변수에 할당한다는 의미입니다.

M-05. 데이터 처리 핵심: Numpy

Numpy(Numerical Python)는 방정식, 행렬, 이미지를 코드로 표현하고 고속으로 계산하기 위한 파이썬 필수 라이브러리입니다. (6강) Numpy.pdf]

  • 일반 Python List와의 차이: Numpy Array는 연속된 메모리에 데이터를 배치하여 반복문 없이(Vectorization) 매우 빠른 배열 연산을 지원합니다.
  • 행렬의 표현:
    • [9, 6, 5] ➜ 1차원 벡터
    • [[2, 2, 1], [2, -1, 2]] ➜ 2차원 행렬 (6강) Numpy.pdf]

M-07. 데이터 관리: Pandas

Pandas는 Python계의 엑셀로 불리며, 행(Row)과 열(Column)로 이루어진 구조화된 데이터(DataFrame)를 처리합니다. (7-1강) Pandas 1.pdf]

핵심 연산: Groupby (7-2강) Pandas 2.pdf]

데이터 분석 시 가장 많이 쓰이는 Groupby 기능은 특정 컬럼을 기준으로 데이터를 묶어 통계를 냅니다. 이 과정은 다음 3단계로 이루어집니다.

  1. Split: 기준에 따라 데이터를 여러 그룹으로 분할
  2. Apply: 각 그룹별로 합(sum), 평균(mean) 등의 함수 적용
  3. Combine: 적용된 결과를 하나의 표로 다시 결합

M-09. AI 프레임워크: PyTorch 기초

PyTorch는 딥러닝 모델 구축을 위한 핵심 도구입니다. 데이터의 기본 단위로 Numpy의 Array와 유사한 Tensor(텐서)를 사용합니다. (1강) PyTorch Intro.pdf]

  • 이미지와 3-D Tensor: 컴퓨터가 보는 컬러 이미지(RGB)는 3차원 텐서로 표현됩니다.
    • 형태: [채널(R,G,B), 높이(Height), 너비(Width)] 차원 (1강) PyTorch Intro.pdf]
  • GPU 할당: 모델을 빠르게 학습시키려면, 생성된 텐서를 컴퓨터의 그래픽카드(GPU) 메모리로 복사하여 병렬 연산을 수행해야 합니다. (3강) Creating Tensors.pdf]

M-11. 기계학습 모델 맛보기

드론 AI에 쓰이는 딥러닝 모델의 두 가지 기본 형태입니다.

  • 선형 회귀 (Linear Regression):
    • 목표: 데이터의 선형 관계($y = wx + b$)를 찾아 '연속적인 숫자'를 예측. (7강) Linear Regression1.pdf]
    • 학습 방법: 오차를 나타내는 손실 함수(Loss Function, 예: MSE)의 값이 최소가 되도록 가중치($w$)와 편향($b$)을 경사하강법으로 수정. (7강) Linear Regression1.pdf]
  • 이진 분류 (Binary Classification):
    • 목표: 데이터가 A인지 B인지(1 또는 0) '두 가지 중 하나'로 분류. (9강) Binary Classification1.pdf]
    • 특징: 모델의 마지막에 Sigmoid(시그모이드) 함수를 통과시켜, 결과값을 0~1 사이의 확률로 변환함. (9강) Binary Classification1.pdf]
