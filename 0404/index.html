M-01. 필수 도구: Jupyter Notebook

우리가 앞으로 코드를 작성하고 실습할 Jupyter Notebook의 핵심 특징입니다.

  • 대화형 코딩: 코드 조각(셀) 단위로 즉시 실행하고 결과를 확인할 수 있어, AI/데이터 분석에 최적화되어 있습니다.
  • 문서화: 코드, 결과(그래프), 설명(마크다운)을 한 문서에 담을 수 있습니다.

🔑 핵심 단축키 (필수!)

  • Shift + Enter: 현재 셀 실행 및 다음 셀로 이동
  • Esc + A: 현재 셀 위에 셀 추가 / Esc + B: 아래에 셀 추가
  • Esc + DD: 현재 셀 삭제

M-02. 파이썬 기초 복습: 변수 및 제어문

프로그램의 흐름을 조종하는 핵심 문법입니다.

  • 자료형 (Data Type): 정수(int), 실수(float), 문자열(str), 리스트(list)
  • 조건문 (Conditionals): if / else를 사용하여 상황에 맞는 명령을 실행. (예: 드론이 사람을 보면 추적, 아니면 정지)
  • 반복문 (Loops): for를 사용하여 코드를 여러 번 실행.
drones = ["Tello", "DJI", "Parrot"]
for drone in drones:
    if drone == "Tello":
        print("Tello 비행 준비 완료")

M-03. 오후 실습 핵심: Numpy & Pandas

데이터 처리를 위해 파이썬이 제공하는 강력한 라이브러리입니다.

  • Numpy (차원 데이터의 왕):
    • 고성능 다차원 배열(행렬)을 생성합니다. (np.array())
    • 드론 카메라의 이미지(픽셀)를 AI가 이해할 수 있는 숫자 행렬로 변환합니다.
  • Pandas (데이터 관리의 엑셀):
    • 행과 열로 구성된 DataFrame(표) 구조로 데이터를 관리합니다.
    • 다양한 실험 결과나 드론 상태 로그를 정리할 때 유용합니다.

🔥 M-04. 딥러닝의 핵심 무기: PyTorch

PyTorch(파이토치)는 전 세계의 AI 연구자와 개발자가 가장 많이 사용하는 딥러닝 프레임워크입니다.

  • Tensor (텐서): PyTorch의 핵심 데이터 구조입니다. Numpy의 배열과 매우 유사한 다차원 행렬입니다.
  • 왜 Numpy 대신 Tensor를 쓸까요?
    • 텐서는 컴퓨터의 그래픽 처리 장치인 GPU에 올라가서 초고속 병렬 연산이 가능합니다!
    • 인공신경망 역전파(Backpropagation) 시 '자동 미분'을 지원합니다.

M-05. PyTorch Tensor 코딩 실습

Jupyter Notebook에서 텐서를 직접 생성해 봅니다.

import torch

# 1. 특정 값으로 텐서 생성
t1 = torch.tensor([[1, 2], [3, 4]])
print(t1.shape) # 크기 확인: torch.Size([2, 2])

# 2. 0 또는 난수로 채워진 텐서 생성
t_zeros = torch.zeros(2, 3) # 2행 3열의 0 행렬
t_rand = torch.rand(2, 2)   # 0~1 사이 난수 행렬

# 3. 텐서를 GPU로 보내기! (매우 중요)
t_gpu = t1.to('cuda') # CPU 메모리 -> GPU 메모리로 이동

M-07. 기계학습 모델 맛보기

다음 주부터 배울 AI 모델의 목적은 크게 두 가지로 나뉩니다.

  • 선형 회귀 (Linear Regression):
    • 데이터 간의 상관 관계(직선)를 찾아 '연속적인 값(숫자)'을 예측합니다.
    • 예: "배터리 잔량 30%일 때 드론이 비행 가능한 남은 시간은?"
    • Loss Function(손실 함수)을 최소화하도록 경사하강법으로 학습합니다.
  • 이진 분류 (Binary Classification):
    • 입력된 데이터가 'A'인지 'B'인지 두 가지 중 하나로 분류합니다.
    • 예: "카메라에 찍힌 물체가 사람인가, 아닌가?"
    • 최종 결정을 위해 Sigmoid(시그모이드) 함수와 확률을 사용합니다.
